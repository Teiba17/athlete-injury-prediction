# athlete-injury-prediction
Machine learning project predicting athlete injuries using training and physiological metrics.

## Objective

Predict athlete injuries using training and physiological metrics.

## Dataset Features

- Age
- Gender
- Height
- Weight
- Training Intensity
- Recovery Days
- Heartbeat
- Fatigue Score

## Exploratory Data Analysis

Findings:

- Female athletes had a slightly higher injury rate.
- Age showed weak predictive power.
- No single feature strongly predicted injury.
- Feature importance was distributed across many variables.

## Models

- Random Forest Classifier

## Results

Accuracy: 53%

## Conclusion

The dataset appears to contain weak predictive signal and may be synthetic. No dominant predictor was identified.
